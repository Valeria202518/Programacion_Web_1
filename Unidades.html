Unidades de Medida en CSS: Absolutas y Relativas

Unidades de Medida Absolutas

Las unidades absolutas son fijas y no cambian en relación con otros elementos. Son útiles cuando se necesita un control preciso sobre el tamaño, pero pueden presentar problemas en dispositivos con diferentes tamaños de pantalla o resoluciones.

Principales unidades absolutas:

  • px (píxeles) - 1px equivale a un punto en la pantalla
  • pt (puntos) - 1pt = 1/72 de pulgada (común en impresión)
  • pc (picas) - 1pc = 12 puntos
  • in (pulgadas) - 1in = 2.54cm
  • cm (centímetros) - Centímetros reales
  • mm (milímetros) - Milímetros reales

Ejemplos de unidades absolutas:

Este párrafo usa píxeles (px): font-size: 16px; padding: 10px;

Este párrafo usa centímetros (cm): font-size: 0.5cm; padding: 0.3cm;

Este párrafo usa puntos (pt): font-size: 12pt; padding: 8pt;

Consejo: Las unidades absolutas son mejores para medios de impresión donde el tamaño físico es conocido. Para pantallas, especialmente en diseños responsivos, las unidades relativas suelen ser más apropiadas.

Unidades de Medida Relativas

Las unidades relativas definen su valor en relación con otro elemento (como el tamaño de fuente del elemento padre o el tamaño de la ventana gráfica). Esto las hace ideales para diseños responsivos que deben adaptarse a diferentes dispositivos y tamaños de pantalla.

Principales unidades relativas:

  • em - Relativo al font-size del elemento padre
  • rem - Relativo al font-size del elemento raíz (html)
  • % (porcentaje) - Relativo al elemento padre
  • vw (viewport width) - 1% del ancho de la ventana gráfica
  • vh (viewport height) - 1% del alto de la ventana gráfica
  • vmin - 1% de la dimensión más pequeña de la ventana gráfica
  • vmax - 1% de la dimensión más grande de la ventana gráfica
  • ex - Altura de la "x" en la fuente actual
  • ch - Ancho del carácter "0" en la fuente actual
  • lh - Altura de línea del elemento

Ejemplos de unidades relativas:

Este párrafo usa em: font-size: 1.2em; padding: 1em;

Anidado con font-size: 0.8em - nota cómo afecta a los em internos

Este párrafo usa rem: font-size: 1.2rem; padding: 1rem;

Anidado con font-size: 0.8rem - los rem siempre son relativos al elemento raíz

Este párrafo usa porcentajes (%): font-size: 110%; padding: 5%; width: 80%;

Este párrafo usa vw y vh: font-size: 2vw; padding: 1vh; width: 50vw;

Redimensiona la ventana del navegador para ver cómo cambia.


Este párrafo usa ex: font-size: 2ex; padding: 1ex;

Este párrafo usa ch: width: 20ch; (Ancho de 20 ceros)

Este párrafo usa lh: font-size: 1lh; (igual a la altura de línea)

Este párrafo usa vmin: font-size: 3vmin; padding: 2vmin;

Este párrafo usa vmax: font-size: 3vmax; padding: 2vmax;

Consejo: Para diseños responsivos modernos, combina rem para tamaños de fuente y márgenes/paddings con unidades de viewport (vw/vh) para elementos que deben escalar con el tamaño de la pantalla.
